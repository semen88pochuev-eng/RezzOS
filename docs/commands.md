## Package Manager
| Command | Description |
|---------|----------|
| `pkg install <name>` | Install package |
| `pkg search <name>` | Search for packages |
| `pkg list` | List installed packages |
| `pkg upgrade` | Upgrade all packages |
| `pkg info <name>` | Package information |

## Services (runit)
| Command | Description |
|---------|----------|
| `rsv list` | List services |
| `rsv up <name>` | Start service |
| `rsv down <name>` | Stop service |
| `rsv restart <name>` | Restart service |

## Development
| Command | Description |
|---------|----------|
| `tcc <file.c>` | Compile C code |
| `lua5.3 <file.lua>` | Run Lua script |
| `make` | Build projects |
